jQuery Shuffle Text Demo

ID指定

指定したIDの要素だけシャッフルします

var shuffle = $("#test").shuffleText();
shuffle.start();

Class指定

Class指定したすべての要素を同時にシャッフルします

var shuffle = $(".test").shuffleText();
shuffle.start();

Class指定で個別シャッフル

Class指定でシャッフルイベントをまとめて付けるけど、シャッフルは個別に行います

var shuffle = $(".test").shuffleText();
shuffle.start(jQueryObject);
